IT-R Class Manager

The online help manual is currently a work in progress.

What is IT-R?

IT-R is a collection of bash scripts and R functions that allow instructors who use servers for teaching with R-Studio to more easily handle tasks such as assigning, collecting, and returning assignments.

Setup Environment

Login and clone the IT-R package from this GitHub Repository.

SSH (preferred)

git clone git@github.com:cameverett/itr.git
or...

HTTPS

git clone https://github.com/cameverett/itr.git

Setting Up Commands

cd into the repository and run bash setup.sh in the terminal.

It's as simple as that! Run classman help to check if the scripts have been setup properly. You should see a man page pop up in your terminal.

If you do not see the man page and receive a message saying a script cannot be found you will need to run source $HOME/.bash_aliases.

Instructor Toolkit

Assign

Copy a file or folder and place it in /home/student/assignments

classman assign -s </path/to/students.txt> -a </path/to/fileorfolder>

Collect

Find and copy student work into your /home/<instructor>/homework folder. The program will find files according to the tag (-t) option given.

classman collect -s </path/to/students.txt> -t <tag>

Create

Creates directories for students /home/<student>/{submit,returned,mynotes}.

classman create -s </path/to/students.txt>

Return

Copies files placed in /home/<instructor>/return folder to your students' /home/<student>/returned folder.

classman return -s </path/to/students.txt> -f <flag>
