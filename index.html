IT-R Class Manager

The online help manual is currently a work in progress.

What is IT-R?

IT-R is a collection of bash scripts and R functions that allow instructors who use servers for teaching with R-Studio to more easily handle tasks such as assigning, collecting, and returning assignments.

System Admins

Setup Environment

Login as a user with sudo privileges. Then clone the GitHub Repository on your machine.

SSH (preferred)

git clone git@github.com:cameverett/itr.git

or...

HTTPS

git clone https://github.com/cameverett/itr.git

Create a text file with the list of all instructor usernames and another text file for all student usernames you want to setup in the test environment and cd into the cloned itr repository. 

(e.g. /home/user/instructors.txt)

instructor0
instructor1
instructor2
instructor3
instructor4

(e.g. /home/user/students.txt)

student0
student1
student2
student3
student4

and then cd /path/to/repo/itr.

Now it is time to run the setup.sh script in the repository.

ls .
sudo bash setup.sh -i /path/to/instructorfile.txt -s /path/to/studentsfile.txt -p bash/

Users will be created with their username in the text file and a password with a 1 added on to the end of the username so that
username: instructor0, password: instructor01

You have now created a test environment of students and teachers. The instructors are given sudo privileges and are part of the instructors user group. Students are treated as regular users with limited permissions on the server.

Reset Environment

Create a text file with the list of all instructor usernames and another text file for all student usernames you want to remove in the test environment and cd into the cloned itr repository. 

(e.g. /home/user/instructors.txt)

instructor0
instructor1
instructor2
instructor3
instructor4

(e.g. /home/user/students.txt)

student0
student1
student2
student3
student4

and then cd /path/to/repo/itr.

Warning This will permanently remove users in these files from the system.

sudo bash cleanup.sh -i </path/to/instructorsfile.txt> -s </path/to/studentsfile.txt>

Instructor Toolkit

Assign (work in progress)

Copy a file or folder and place it in /home/student/class/assignments folder.

class_assign -c <class> -s </path/to/studentfile>

Collect (ready for testing)

Find and copy student work and copy the files into your /home/<instructor>/<class>/submissions folder. The program will find files according to the tag (-t) option given.

class_collect -i <your_username> -c <class> -s </path/to/studentfile>

Create (ready for testing)

Creates directories for students /home/<student>/<class>/{submit,returned,mynotes}.

class_create -i <your_username> -c <class> -s </path/to/studentfile>

Return (work in progress)

Copies files placed in /home/<instructor>/<class>/return folder to your students' /home/<student>/<class>/returned folder.

class_return -i <your_username> -c <class> -s </path/to/studentfile>
